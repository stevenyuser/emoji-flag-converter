Emoji Flag Converter
-----------

Converts a Flag Code or Country Name into a country's flag emoji.

Heavily inspired by: https://github.com/jonathan-kosgei/emoji-flags
 - Differs by allowing country name as well

Usage
```
> flagUS = byCode("US")
> print(flagUS)
> 🇺🇸

> flagCN = byName("China")
> print(flagChina)
> 🇨🇳
```

Hope this helps!
